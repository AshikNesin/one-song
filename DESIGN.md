# Design System

## Color Palette

### Primary Colors

| Color | Hex Code | Usage |
|-------|----------|-------|
| Pure Black | `#000` | Main background (all screens) |
| Pure White | `#fff` | Primary text, active states, buttons |
| Off-White | `#ccc` | Secondary text, inactive options |

### Gray Scale (UI Elements)

| Color | Hex Code | Usage |
|-------|----------|-------|
| Dark Gray 1 | `#1a1a1a` | Cards, containers, artwork placeholder |
| Dark Gray 2 | `#2a2a2a` | Borders, dividers, inactive chips |
| Dark Gray 3 | `#333` | Buttons, inactive track backgrounds |
| Medium Gray | `#666` | Arrow icons, tertiary elements |
| Light Gray | `#999` | Subtitle text, artist names, secondary labels |
| Very Light Gray | `#555` | Footer text |

### Semantic Colors

| Color | Hex Code | Usage |
|-------|----------|-------|
| Error Red | `#ff4444` | Error messages, danger actions, destructive buttons |

### Transparent/Overlays

| Color | Code | Usage |
|-------|------|-------|
| Modal Overlay | `rgba(0,0,0,0.7)` | Modal backdrop |
| Loading Indicator | `rgba(255,255,255,0.4)` | Activity indicator |

## Theme Characteristics

This project follows a **dark mode aesthetic** with:

- Black backgrounds (`#000`)
- White text and accents (`#fff`)
- Gray hierarchy for secondary content (`#999`, `#ccc`, `#666`)
- Red for errors/danger (`#ff4444`)

## Logo Guidelines

For brand consistency, logos should use:
- **White on black** or **black on white** for a clean, minimalist look
- Simple, bold shapes that work at small sizes (app icon)
- Avoid colors outside the grayscale palette unless intentionally highlighting a feature
